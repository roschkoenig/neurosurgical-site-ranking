# Neurosurgical Site KOL Ranking

A reproducible Python pipeline that ranks US glioblastoma neurosurgical sites
by literature-based **Key Opinion Leader (KOL) strength**, using PubMed and
OpenAlex as the primary data sources.

---

## Project goal

1. Retrieve a corpus of surgically relevant glioblastoma papers from PubMed.
2. Enrich them with citation and authorship metadata from OpenAlex.
3. Build an author-level co-authorship network and compute influence metrics.
4. Map authors to canonical US trial-site institutions.
5. Compute site-level KOL and cluster-depth scores.

## Setup

### 1. Clone and install

```bash
git clone https://github.com/roschkoenig/neurosurgical-site-ranking.git
cd neurosurgical-site-ranking
python -m venv venv && source venv/bin/activate   # optional but recommended
pip install -r requirements.txt
```

### 2. API notes

| Service   | Key required? | How to configure |
|-----------|---------------|-----------------|
| PubMed    | No (low volume) / optional for higher throughput | `export NCBI_API_KEY=<key>` |
| OpenAlex  | No | Set `export OPENALEX_EMAIL=you@example.com` to join the polite pool |
| OpenAI (optional LLM step) | Yes | `export OPENAI_API_KEY=<key>` |

The pipeline works without any API keys.  Adding an NCBI key raises the
E-utilities rate limit from 3 to 10 requests/second.

### 3. Configure site aliases

Edit `data/site_aliases.csv` to add or remove sites and their aliases:

```csv
canonical_site,alias
UCSF Medical Center,UCSF
UCSF Medical Center,University of California San Francisco
...
```

## Execution

### Run in Google Colab

Open `notebooks/pipeline_colab.ipynb` in Colab.  All steps are documented
inline.  Results are automatically downloaded at the end.

### Run locally (script-style)

```python
from src.pubmed_search   import PubMedSearcher
from src.openalex_client import OpenAlexClient
from src.paper_classifier import PaperClassifier
from src.author_network  import AuthorNetwork
from src.site_matcher    import SiteMatcher
from src.site_scorer     import SiteScorer

# 1. Retrieve papers
searcher = PubMedSearcher()
pmids    = searcher.search(max_results=500)
papers   = searcher.fetch_details(pmids)

# 2. Enrich with OpenAlex
papers = OpenAlexClient().enrich_papers(papers)

# 3. Classify
papers = PaperClassifier().classify_all(papers)

# 4. Author network
net = AuthorNetwork()
net.build(papers)
authors_df = net.compute_metrics()

# 5. Site matching
matcher    = SiteMatcher("data/site_aliases.csv", use_llm=False)
authors_df = matcher.match_authors(authors_df)
net.save_authors_csv(authors_df)
matcher.save_audit()
matcher.save_unresolved(authors_df)

# 6. Site scoring
site_scores = SiteScorer().compute(authors_df)
SiteScorer().save(site_scores)
```

## Surgical relevance classifier

Papers are labelled as:

| Label | Weight | Description |
|-------|--------|-------------|
| `core_surgical` | 3.0 | Directly about GBM surgery / operative technique |
| `surgery_adjacent` | 1.0 | GBM treatment adjacent to surgery |
| `non_core` | 0.0 | Biology, preclinical, paediatric, etc. |

Rules operate on title, abstract, journal name, and concept keywords.
See `src/paper_classifier.py` for the full rule set.

---

## Site matching

Affiliation strings are matched in order:

1. **Exact alias** – normalised alias substring match (`match_method = exact_alias`)
2. **Fuzzy match** – token-set-ratio via `rapidfuzz` (`match_method = fuzzy`)
3. **LLM adjudication** (optional) – only for low-confidence matches (`match_method = llm`)
4. **Unresolved bucket** – all remaining (`match_method = unresolved`)

Confidence and match method are recorded in `match_audit.csv` for every
affiliation.

---

## Caching

All API responses (PubMed + OpenAlex) are cached locally in the `cache/`
directory as JSON files keyed by a SHA-256 hash of the request parameters.
Re-running the pipeline is fast and free after the first run.

---

## Extending the pipeline

- **Better classifier** – replace `PaperClassifier.classify()` with a learned
  model (e.g. fine-tuned BERT); the interface takes a paper dict and returns a
  label string.
- **More sites** – add rows to `data/site_aliases.csv`.
- **Citation influence edges** – add cited-by links between authors in
  `AuthorNetwork.build()` using the `referenced_works` field from OpenAlex.
